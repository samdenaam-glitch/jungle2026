JUNGLE QUEST 2026 - HTML5 EDITION
================================

A complete quantum-powered platformer that runs in any modern browser.
Inspired by classic MS-DOS games like Jungle Book and Rayman.

HOW TO PLAY:
------------

1. Save all files in the same folder
2. Open index.html in Chrome/Firefox/Edge
3. Click "NEW GAME" to start
4. Use arrow keys or WASD to move and jump
5. Collect bananas for score, keys to progress
6. Use quantum abilities with 1, 2, 3 keys
7. Complete all 4 levels to win!

CONTROLS:
---------

Keyboard:
- Arrow Keys / WASD: Move and jump
- Space: Jump
- 1: Quantum Scan (reveals collectibles)
- 2: Time Jump (switch timeline)
- 3: Quantum Entangle (link collectibles)
- ESC: Pause game
- Enter/Space: Menu selection

Mobile/Touch:
- On-screen buttons appear automatically
- Touch menu items to select

CHEAT CODES (for testing):
--------------------------
- Ctrl+Q: Add 1000 score
- Ctrl+T: Switch timeline
- Ctrl+B: Add 10 bananas
- Ctrl+L: Add 3 lives
- Ctrl+N: Skip to next level
- Shift+D: Toggle debug mode

TROUBLESHOOTING:
----------------
1. If game doesn't load, check browser console (F12)
2. Make sure all files are in the same folder
3. Try a different browser if audio doesn't work
4. On mobile, ensure "Request Desktop Site" is off
